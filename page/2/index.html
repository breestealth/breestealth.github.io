使用OpenWRT来科学上网

背景：使用shaodowsocks，非VPN方式。

方案一：国内流量不走代理，其余的全部走代理。

  1. 下载shadowsocks for openwrt的spec版本并安装配置；
  2. 安装ChinaDNS，将上级查询地址设为114.114.114.114,127.0.0.1:5300；
  3. 根据文档，配置好dnsmasq作为本地域名解析服务；
  4. 启动shadowsocks/ChinaDNS以及dnsmasq；
  5. DONE

方案二：根据GFWList作为依据进行代理

  1. 下载shadowsocks for openwrt普通版本并安装配置。同时将/etc/init.d/shadowsocks中的启动命令改为ss-redir和ss-tunnel，并配置好ss-tunnel的转发规则（建议转发到5300端口的数据直接转发到8.8.8.8:53，也就是作为DNS域名解析的代理）；
  2. 安装ChinaDNS，将上级查询地址设为114.114.114.114,127.0.0.1:5300；
  3. 卸载dnsmasq并安装dnsmasq-full、ipset以及iptables-mod-nat-extra三个组件；

  1. 修改/etc/dnsmasq.conf文件，添加一行conf-dir=/etc/dnsmasq.d。同时在/etc目录下创建dnsmasq.d的文件夹；
  2. 使用脚本生成GFWList。请根据需要修正DNS的端口，若没有修正ChinaDNS的默认配置，该数值为5353；

  4. 重启路由器；
  5. Done

几个需要注意的点：

  1. ss-tunnel的作用是转发DNS请求。国内的墙会对DNS进行投毒，所以通过代理使用Google提供的DNS进行解析，从而返回正确的结果；
  2. ss-redir的目的是为了流量转发，将特定标识的流量（本例中为需要X墙的流量使用Shadowsocks建立的代理转发，从而获取数据）
  3. shadowsocks建议购买而不是自行搭建，速度上面无法保证。如果需要建立，可以使用这个链接进行购买最低配的VPS，一年4刀，足够一个人使用；

看图说故事（119）

我去拜访我的老朋友，C，他住在后面的山头上面。

我辛辛苦苦走到了半山腰，看到了这只狗——纯白的一只。它虎视眈眈的看着我，仿佛想从我的行为中看出我的意图，那种不友好的眼神，让我有种不自在的感觉。

当我准备绕过去的时候，它走了两部，挡在了我的面前；我换了个方向，它又跑过来挡住我。

如此总共三次。

我算是看出来了，它就是故意不让我过去！！！

无奈只能打电话给C，告诉他这个情况。

C在电话中哈哈大笑，然后说“你把外放打开，我和它说两句。”

C在电话那头对这只狗说了两句话，狗狗乖乖的让开，再也没有挡住我。

见到C，我一把拉住他——“快和我说说到底怎么回事。”

原来这只狗是C收养的流浪狗，在它快要饿死的时候，C给了它一些吃的。就这么着，它活了下来，也逐渐就将C当作了主人，起到了看家护院的责任。这些都不是C教它的，而是它自己学会的。对于这点，C也表示很惊讶。究其原因，C觉得可能是本能或者动物与动物之间那种纯真的信任所起。

等我下山，那只狗也没有再拦我，而是欢快的摇起了尾巴送我下山。

我相信那一刻，它对我的感觉，也一定是信任。

看图说故事（118）

这里是一个难民区，所有的居住者都是因为战争而背井离乡的难民。这里臭气熏天，遍地垃圾。

大家应该都觉得这里一定是个悲伤的地方。

可它真的不是。

在这里我听到了比别的地方多很多的欢声笑语，也看到了非常多的关爱。

因为这里有非常多的孩子。

在这里，孩子的比例占到了70％，满大街都是孩子。而纯真无邪的孩子，将这里点缀成了一片欢乐的海洋。

在这里，孩子们赤裸着上身，满大街的跑着、嚷着，互相打招呼，在下雨天，还会用脚踢起那一片的水花。

还有那些孩子的笑容，很美。

看图说故事（116）

Jim原本是一名士兵，直到他因为战斗伤残后退伍回家。

虽然拿到了一笔不菲的补偿金，但是坐吃山空总有那么一天，未雨绸缪才是王道。

于是Jim做起了个小生意——卖伞。

虽说着都是卖伞，但是伞和伞不一样。Jim在部队中属于拆弹部队，所以手非常精巧，可以做很多精巧的工具。Jim也将这个优势带到了日常生活中。

他虽然也是弄过来一批伞卖，但是每把伞Jim都会拆开做一些二次加工，比如加一些支撑物啦，做一些机械上的优化之类。经过Jim改造的伞，往往使用寿命和使用体验都有非常明显的提升。

随之而来的就是Jim的伞卖的非常好。

这天我又来到了Jim卖伞的地方。这个家伙身边摆着一堆伞，自己撑着把小伞已经睡着了——看样子昨天又忙到很晚。我不忍心打扰他，只是默默的写下这篇文章，希望能帮他多多宣传。

需要伞的话，就去XX大街买一把吧，你一定不会失望哦。

看图说故事（115）

要是问对于男人来说，最痛苦的是啥，那一定是陪人逛街，而且绝对没有之一。

比如你看Lee。

Lee是某IT公司高管。他往常工作的时候，英姿飒爽，杀伐决断毫不犹豫。那个时候的他，一副“天下尽在我手”的霸气。在这种霸气之下，他帮助公司完成了好几论融资，最终帮公司顺利上市，成为了行业内的领头羊。

当然，领头羊的高管也是有弱点的。

比如现在，我就在他的对面，看着他在Dior门店外面，目光呆滞的插手而立，不知所措。

没错，你说对了——他老婆在里面选购，他立马表现出了所有男人通用的“好无聊”的表情，而后成功的被“发配”到了外面等待。

我上前去给Lee打了个招呼，他马上从发呆中醒过来，立马又恢复了公司中的状态。我也没有说破，只是淡淡一笑，递过去一根香烟，他接了过去，掏火机、点上、吐烟圈，一气呵成。

看，男人果然都是一个样。

看图说故事（113）

M和W是一对恋人。

他们已经活了500年，而他们的长相依然是20出头的样子——换言之，他们不会老，更不会死。

这个秘密就是他们之间能相互产生生命之火。只要生命之火存在，他们就永远不会老。

但是他们已经厌倦了这种不死的生活。他们看着人类的发展，看到了森林如何的被砍伐、动物如何的被猎杀、环境如何的被破坏。

他们也偷偷做了很多，但是在人类前进的车轮前，他们的努力非常渺小。他们逐渐开始心灰意冷，甚至已经不愿意看着这个世界的无情的变化。

“我不想活了。”有一天M对W说。

“我也是。”W回答。

三天后，他们来到了最后一片雨林，他们选中了森林中间一棵树。他们分立在树的两旁，相视一笑。

“我们不忍心看着地球绿色如此消逝。只能将最后希望寄托给你，希望你可以替我们守护这片森林，拜托了。”

生命之火灌注到了这棵树上，慢慢的这棵树开始快速生长，不久就突破了这片雨林，仿佛一个守护者俯身看着这片雨林。而M和W两人，形容枯槁，拉着手坐在了这棵守护者的脚下，很安详的，去了。

看图说故事（112）

Jack坐在了墙边，等着最后一缕阳光的消失。

他熄灭了烟头，静静坐在那里，不发一言。他心中明白，最后一缕阳光落山后，他的生命也就到了尽头。

他心中淡淡一笑，飘过两个字——无悔。

在杀了最后一名嗜血者后，他自己也被嗜血者咬到。毒素快速在他身体里蔓延，他的生命只到太阳落山前的最后一刻。

他此刻心中只有一念——“我们就要见面了。”

他的爱人，死于嗜血者的攻击——这也是他为什么发誓要消灭所有嗜血者的原因。现在他做到了，但是他的生命也要就此结束——这是否是他的爱人所要的呢？

此时此刻，考虑这些问题已经毫无意义，不如淡淡的坐在这里，等着生命最后一刻的降临。

那个世界的你，会怎么样子呢？会不会比这边要好看？会不会也很思念我？

随着最后一缕阳光的消逝，他身影也消失在黑暗中。剩下的，只有一张椅子。
